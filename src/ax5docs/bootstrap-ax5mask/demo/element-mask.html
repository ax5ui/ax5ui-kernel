Element mask

It can be used when the need to mask the element.

user content

In order to put the mask on, you can use the following example elements and codes 

<div style="width: 300px;height: 300px; border: 1px solid #000;" id="user-content">
    user content
</div>

Javascript 

<script type="text/javascript">
    var mask = new ax5.ui.mask();

    $(document.body).ready(function () {

        $('#mask-open').click(function () {
            mask.open({
                zIndex: 99,
                target: $("#user-content").get(0),
                content: 'MASK'
            });
        });

        $('#mask-close').click(function () {
            mask.close();
        });

    });
</script>
