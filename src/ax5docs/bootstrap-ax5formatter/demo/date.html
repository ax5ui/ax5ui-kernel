Formatter Date

Paste the following code into the head section of your site's HTML.

<script type="text/javascript" src="src/bootstrap-ax5formatter/dist/ax5formatter.js"></script>
HTML 
<div class="form-group">
    <div class="input-group">
        <span class="input-group-addon">Date</span>
        <input name="3" type="text" class="form-control" placeholder="yyyy-mm-dd" data-ax5formatter="date">
    </div>
</div>
<div class="form-group">
    <div class="input-group">
        <span class="input-group-addon">Date(time)</span>
        <input name="3" type="text" class="form-control" placeholder="yyyy-mm-dd hh:mi:ss" data-ax5formatter="date(time)">
    </div>
</div>
<div class="form-group">
    <div class="input-group">
        <span class="input-group-addon">Time</span>
        <input name="3" type="text" class="form-control" placeholder="hh:mi:ss" data-ax5formatter="time">
    </div>
</div>
Date
Date(time)
Time
JS 
<script type="text/javascript">
    $('[data-ax5formatter]').ax5formatter();
</script>
